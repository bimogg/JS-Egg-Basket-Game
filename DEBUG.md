# Как открыть инструменты разработчика в Safari на Mac

## Способ 1: Через меню Safari
1. Откройте Safari
2. В меню сверху выберите **Safari** → **Настройки** (или **Preferences**)
3. Перейдите на вкладку **Дополнения** (или **Advanced**)
4. Поставьте галочку **"Показывать меню "Разработка" в строке меню"** (или **"Show Develop menu in menu bar"**)
5. Теперь в меню сверху появится пункт **Разработка** (или **Develop**)
6. Выберите **Разработка** → **Показать веб-инспектор** (или **Show Web Inspector**)

## Способ 2: Горячие клавиши
1. Сначала включите меню "Разработка" (см. Способ 1, шаги 1-4)
2. Нажмите **Cmd + Option + I** (или **⌘ + ⌥ + I**)

## Что смотреть в консоли:
1. Откройте вкладку **Console** (Консоль)
2. Ищите сообщения:
   - "Фон загружен успешно" - значит фон загрузился
   - "Ошибка загрузки фона" - значит файл не найден
   - Красные ошибки - проблемы с загрузкой

## Что смотреть в Elements (Элементы):
1. Выберите элемент `<body>` или `<html>`
2. В правой панели смотрите стили
3. Проверьте, есть ли `background-image: url('imgs/background.jpg')`
4. Если есть красная линия через свойство - значит оно переопределено или не работает

## Проверка загрузки файла:
В консоли выполните:
```javascript
new Image().src = 'imgs/background.jpg'
```
Если файл загружается, вы увидите его в Network (Сеть) вкладке.

## Network (Сеть) вкладка:
1. Откройте вкладку **Network** (Сеть)
2. Обновите страницу (Cmd+R)
3. Ищите файл `background.jpg`
4. Если он красный - файл не загрузился
5. Если зеленый - файл загрузился, но возможно проблема в CSS
